# Backend.AI UI

A collection of reusable React components for Backend.AI WebUI.

## Our vision

The goal is to create a component library that makes it easier to build applications with a consistent look and feel, whether you're contributing to Backend.AI WebUI or developing plugins and extensions.

With this library, we aim to:

- **Understand component usage**: See how each component works through interactive examples
- **Develop faster**: Use pre-built, production-ready components
- **Maintain consistency**: Ensure visual and behavioral consistency across the application
- **Type safety**: Full TypeScript support for better developer experience

## Getting started

1. **Explore**: Browse the components in the sidebar to discover what's available
2. **Interact**: Use the Controls panel to experiment with different props and states
3. **Implement**: Copy the code examples into your project

## Installation

```bash
pnpm add backend.ai-ui
```

```bash
npm install backend.ai-ui
```
