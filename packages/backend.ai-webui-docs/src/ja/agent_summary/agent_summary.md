# エージェントサマリー

エージェントサマリーページでは、プロジェクトで利用できるコンピュート**エージェント**（エージェントノード）と、各エージェントのリソース割り当て状況を読み取り専用で確認できます。コンピュートセッションを作成する前に、CPU、メモリ、アクセラレータの残り容量を確認するために使用します。

このページは、サーバーの `hideAgents` 設定が無効になっている場合に **Metrics** メニューグループに表示されます。22.09以降、Backend.AI WebUI はサーバーの設定に応じて、エージェントノードの一部情報を一般ユーザーにも表示できます。

## フィルタリングとステータス

リストの上部にあるツールバーで、表示するエージェントを絞り込めます。

- **ステータス切り替え**: **接続済み**と**終了しました**のエージェントを切り替えます。デフォルトでは**接続済み**のエージェントが表示されます。
- **ID**: エージェント ID でリストをフィルタリングします。
- **スケジューリング可能かどうか**: エージェントが新しいセッションを受け入れられるかどうかでフィルタリングします。**有効**または**無効**を選択します。
- **更新**: **更新**ボタンをクリックすると、最新のエージェントデータでリストを再読み込みします。

## 列

エージェントリストには次の列が表示されます。

- **ID**: エージェントの一意の識別子です。この列は並べ替えできます。
- **アーキテクチャ**: エージェントの CPU アーキテクチャです（例: `x86_64`）。
- **割り当て情報**: エージェントの各リソーススロットの使用中の容量と利用可能な容量を、使用率のプログレスバーとともに表示します。
   * CPU はコア数、メモリは GiB 単位で表示され、エージェントに搭載されているアクセラレータ（GPU や fGPU など）は追加のスロットとして表示されます。
   * 使用率が 80% を超えると、プログレスバーが赤色で表示されます。
- **リソースグループ**: エージェントが属するリソースグループです。この列は並べ替えできます。
- **スケジューリング可能かどうか**: エージェントが新しいセッションを受け入れられるかどうかです。チェックアイコンはスケジューリング可能であることを、淡色表示のマイナスアイコンは不可であることを示します。この列は並べ替えできます。

表の右下にある列設定の歯車アイコンを使用して、表示する列とその順序をカスタマイズできます。非表示にした列と列の順序は、次回アクセス時にも保持されます。

:::note
SFTP 専用のリソースグループに属するエージェントは、このリストから自動的に除外されるため、ストレージ専用のエージェントはここには表示されません。
:::

セッションを開始する前にリソース割り当てを計画するには、ここで利用可能な容量を確認してから[セッションページ](#session-page)に進んでください。
