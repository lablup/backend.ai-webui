# ข้อตกลงการอนุญาตใช้งาน

## ใบอนุญาต Backend.AI (ซอฟต์แวร์)

เอกสารนี้กำหนดเงื่อนไขของข้อตกลงใบอนุญาตสำหรับซอฟต์แวร์ Backend.AI ค่าธรรมเนียมการใช้งานและแผนสนับสนุนบริการ Backend.AI Cloud ที่จัดทำโดย Lablup เป็นอิสระจากนโยบายนี้

**คำจำกัดความ**

- ฮาร์ดแวร์: รวมถึงคอมพิวเตอร์ทางกายภาพ, เครื่องเสมือน และสภาพแวดล้อมของคอนเทนเนอร์ที่ผู้ใช้เป็นเจ้าของหรือเช่าและมีสิทธิในการเรียกใช้ซอฟต์แวร์
- องค์กร: บุคคล, บริษัท, องค์กร, สถาบัน (รวมถึงองค์กรไม่แสวงหาผลกำไรและองค์กรพาณิชย์; อย่างไรก็ตาม บริษัทลูกที่เป็นร่างกายแยกต่างหากไม่รวมอยู่ด้วย)

LGPL ต้องได้รับการปฏิบัติตามเมื่อผู้ใช้ใช้งานและเปลี่ยนแปลง Backend.AI Server (Manager / Agent / Common) หรือพัฒนาและจัดจำหน่ายซอฟต์แวร์ที่ใช้มัน อย่างไรก็ตาม ในกรณีที่จัดจำหน่ายซอฟต์แวร์ที่นำเข้า Backend.AI server เป็นโมดูลโดยไม่เปลี่ยนแปลง (เช่น Python import) จะถือว่าเป็นการเชื่อมโยงแบบไดนามิก และการเปิดเผยโค้ตตาม LGPL จะไม่บังคับ เมื่อ Backend.AI server ถูกติดตั้งบนฮาร์ดแวร์และใช้งานโดยประชาชนทั่วไปผ่านเครือข่าย จะไม่มีข้อผูกพันตาม LGPL

การตีความที่ถูกต้องของกรณีอื่น ๆ ทั้งหมดอยู่ภายใต้ข้อความต้นฉบับของ LGPL และคำพิพากษาของศาล

นอกเหนือจากการปฏิบัติตาม LGPL การทำสัญญาทางการค้าจะต้องทำกับ Lablup ในกรณีต่อไปนี้:

#. เมื่อซอฟต์แวร์ที่ทำงานได้เฉพาะหลังจากการติดตั้งเซิร์ฟเวอร์ Backend.AI ถูกขายให้กับลูกค้านอกองค์กร
#. เมื่อฮาร์ดแวร์รวมถึงเซิร์ฟเวอร์ Backend.AI ถูกขายให้กับลูกค้านอกองค์กร
#. เมื่อเซิร์ฟเวอร์ Backend.AI ถูกติดตั้งในฮาร์ดแวร์และได้รับค่าธรรมเนียมการใช้งานจากลูกค้านอกองค์กรที่ใช้งานมัน

ในกรณีอื่น ๆ คุณสามารถใช้เซิร์ฟเวอร์ Backend.AI ได้ฟรี

**ตัวอย่างการตีความ**

- หากคุณกระจายเซิร์ฟเวอร์ Backend.AI ที่มีการปรับเปลี่ยนไปยังภายนอกองค์กร คุณต้องเปิดเผยโค้ดและใช้ LGPL ในลักษณะเดียวกัน ไม่มีข้อผูกพันในการเปิดเผยโค้ดหากใช้ภายในองค์กรเท่านั้น
- เมื่อมีการจัดจำหน่ายซอฟต์แวร์ที่ใช้เซิร์ฟเวอร์ Backend.AI เป็นห้องสมุดที่จำเป็น

  - การแจกจ่ายฟรี: ซอฟต์แวร์ไม่จำเป็นต้องเป็น (L)GPL และไม่ต้องมีสัญญาแยกต่างหากกับ Lablup
  - การแจกจ่ายแบบชำระเงิน: ซอฟต์แวร์ไม่จำเป็นต้องเป็น (L)GPL แต่จำเป็นต้องมีสัญญาการค้าที่ระหว่าง Lablup

- เมื่อเซิร์ฟเวอร์ Backend.AI ถูกติดตั้งบนฮาร์ดแวร์และมีการใช้งานโดยสาธารณะผ่านเครือข่าย

  - การแจกจ่ายฟรี: ไม่จำเป็นต้องมีสัญญาแยกต่างหากกับ Lablup
  - การจัดจำหน่ายที่ต้องชำระเงิน: จำเป็นต้องมีสัญญาพาณิชย์กับ Lablup

- เมื่อกระจายฮาร์ดแวร์ที่ติดตั้ง Backend.AI เซิร์ฟเวอร์อยู่

  - การแจกจ่ายฟรี: ไม่จำเป็นต้องมีสัญญาแยกต่างหากกับ Lablup
  - การจัดจำหน่ายที่ต้องชำระเงิน: จำเป็นต้องมีสัญญาพาณิชย์กับ Lablup

สัญญาการค้าที่รวมถึงค่าธรรมเนียมการสมัครสมาชิกรายเดือน / รายปีสำหรับเวอร์ชันองค์กรโดยค่าเริ่มต้น แต่รายละเอียดอาจแตกต่างกันไปขึ้นอยู่กับสัญญาเฉพาะบุคคล
