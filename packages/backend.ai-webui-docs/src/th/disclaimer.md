# คำชี้แจง

ข้อมูลและเนื้อหาในคู่มือนี้มีให้ในลักษณะ "ตามที่เป็น" และ "ตามที่มี" เพื่อวัตถุประสงค์ในการให้ข้อมูลและอ้างอิงเท่านั้น โดยไม่มีการรับประกันใด ๆ ทั้งแบบชัดแจ้งหรือโดยนัย รวมถึง แต่ไม่จำกัดเพียง การรับประกันที่โดยนัยเกี่ยวกับการค้า ความเหมาะสมสำหรับวัตถุประสงค์เฉพาะ และการไม่ละเมิด Lablup Inc. ไม่มีความรับผิดชอบต่อความเสียหายใด ๆ ที่เกิดจากการใช้คู่มือนี้ รวมถึงความเสียหายที่เกิดขึ้นตามมา แม้ว่าจะมีความพยายามอย่างมากเพื่อให้แน่ใจในความถูกต้องของข้อมูลที่ให้ไว้ Lablup Inc. ไม่ได้มีการแสดงถึงหรือให้สัญญาหรือการรับประกันเกี่ยวกับความสมบูรณ์ ความถูกต้อง ความทันสมัย หรือความเหมาะสมของข้อมูล และไม่รับผิดชอบต่อข้อมูลที่ล้าสมัยหรือข้อผิดพลาด Lablup Inc. ไม่มีการแสดงถึงหรือการรับประกันทั้งแบบชัดแจ้งหรือโดยนัยเกี่ยวกับความถูกต้องหรือความสมบูรณ์ของข้อมูลที่มีอยู่ในคู่มือนี้

หากคุณเชื่อว่ามีข้อผิดพลาดในเอกสารนี้ที่แตกต่างจากข้อเท็จจริง โปรดติดต่อเรา เราจะตรวจสอบโดยเร็วที่สุดเท่าที่จะทำได้

ผลิตภัณฑ์ในคู่มือนี้อาจมีการเปลี่ยนแปลงโดยไม่ต้องแจ้งล่วงหน้าเนื่องจากการพัฒนาต่อเนื่องโดยชุมชนโอเพ่นซอร์สและบริษัท Lablup Inc.
