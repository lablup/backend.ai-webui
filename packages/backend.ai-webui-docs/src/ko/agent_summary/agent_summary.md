# 에이전트 요약

에이전트 요약 페이지에서는 프로젝트에서 사용할 수 있는 연산 **에이전트**(에이전트 노드)와 각 에이전트의 자원 할당 현황을 읽기 전용으로 확인할 수 있습니다. 연산 세션을 생성하기 전에 CPU, 메모리, 가속기 용량이 얼마나 남아 있는지 확인하는 데 사용합니다.

이 페이지는 서버의 `hideAgents` 설정이 비활성화되어 있을 때 **Metrics** 메뉴 그룹에 표시됩니다. 22.09 버전부터 Backend.AI WebUI는 서버 설정에 따라 에이전트 노드의 일부 정보를 일반 사용자에게도 표시할 수 있습니다.

## 필터링 및 상태

목록 위쪽의 도구 모음에서 표시할 에이전트를 좁힐 수 있습니다.

- **상태 토글**: **연결중**과 **종료됨** 에이전트를 전환합니다. 기본적으로 **연결중** 에이전트가 표시됩니다.
- **ID**: 에이전트 ID로 목록을 필터링합니다.
- **스케줄링 가능**: 에이전트가 새 세션을 받을 수 있는지 여부로 필터링합니다. **활성화** 또는 **비활성화**를 선택합니다.
- **새로고침**: **새로고침** 버튼을 클릭하면 최신 에이전트 데이터로 목록을 다시 불러옵니다.

## 컬럼

에이전트 목록에는 다음 컬럼이 표시됩니다.

- **ID**: 에이전트의 고유 식별자입니다. 이 컬럼은 정렬할 수 있습니다.
- **아키텍처**: 에이전트의 CPU 아키텍처입니다(예: `x86_64`).
- **할당 정보**: 에이전트의 각 자원 슬롯별 사용 중인 용량과 사용 가능한 용량을 사용률 진행 막대와 함께 표시합니다.
   * CPU는 코어 수, 메모리는 GiB 단위로 표시되며, 에이전트에 존재하는 가속기(GPU 또는 fGPU 등)는 추가 슬롯으로 표시됩니다.
   * 사용률이 80%를 초과하면 진행 막대가 빨간색으로 표시됩니다.
- **자원 그룹**: 에이전트가 속한 자원 그룹입니다. 이 컬럼은 정렬할 수 있습니다.
- **스케줄링 가능**: 에이전트가 새 세션을 받을 수 있는지 여부입니다. 체크 아이콘은 스케줄링이 가능함을, 흐리게 표시된 마이너스 아이콘은 불가능함을 나타냅니다. 이 컬럼은 정렬할 수 있습니다.

표 오른쪽 아래의 컬럼 설정 톱니바퀴 아이콘을 사용하여 표시할 컬럼과 순서를 사용자 지정할 수 있습니다. 숨긴 컬럼과 컬럼 순서는 다음 방문 시에도 유지됩니다.

:::note
SFTP 전용 자원 그룹에 속한 에이전트는 이 목록에서 자동으로 제외되므로, 스토리지 전용 에이전트는 여기에 표시되지 않습니다.
:::

세션을 시작하기 전에 자원 할당을 계획하려면, 여기에서 사용 가능한 용량을 확인한 후 [세션 페이지](#session-page)로 이동하세요.
