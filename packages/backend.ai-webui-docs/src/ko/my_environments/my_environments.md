# 나의 실행 환경

24.03 버전부터 사용자를 위한 새로운 "나의 실행 환경" 페이지가 도입되었습니다.
이 페이지에서는 사용자의
[세션 커밋](#save-session-commit)으로
생성된 이미지 목록 등의 내용을 확인할 수 있습니다.

나의 실행 환경 페이지의 이미지 탭에서 사용자는 연산 세션 생성에 사용되는 사용자 정의 이미지를 관리할 수 있습니다. 이 탭은 연산 세션으로부터 변환된 이미지들의 메타 정보가 표시됩니다. 각 이미지에 대한 레지스트리, 아키텍쳐, 네임 스페이스, 언어, 버전, 기반, 제약, digest 등의 정보를 확인할 수 있습니다.

이미지를 삭제하고 싶다면 제어 열의 빨간색 휴지통 버튼을 클릭합니다. 삭제 후에는 해당 이미지로 새로운 세션을 생성할 수 없습니다.

![](../images/my_environments.png)

사용자 정의 이미지를 복사해서 세션을 생성할 수도 있습니다. 제어 열의 '복사' 버튼을 클릭한 뒤, 세션 페이지로 이동해서 세션을 생성합니다. 방금 복사한 이미지를 직접 입력하는 이미지의 입력으로 붙여넣기 하면 됩니다.

![](../images/copy_image_name_manual.png)

특정 열을 숨기거나 보이게 하려면, 테이블 우측 하단의 기어 아이콘을 클릭합니다. 그러면 아래와 같은 다이얼로그가 나타나며, 보고 싶은 열을 선택할 수 있습니다.

![](../images/table_setting.png)
