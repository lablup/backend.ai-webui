# เริ่มต้นใช้งาน

ส่วนนี้ให้ภาพรวมอย่างรวดเร็วเกี่ยวกับวิธีการเริ่มใช้ Backend.AI ไม่ว่าคุณจะกำลังตั้งค่าการติดตั้งใหม่หรือสำรวจการปรับใช้ที่มีอยู่ ขั้นตอนเหล่านี้จะช่วยให้คุณเริ่มต้นและดำเนินการได้

## ภาพรวม

Backend.AI ประกอบด้วยส่วนประกอบหลักหลายส่วน:

- **Manager**: ระนาบควบคุมส่วนกลางที่จัดการทรัพยากรและเซสชัน
- **Agent**: โหนดคอมพิวต์ที่รันคอนเทนเนอร์ (เซสชัน)
- **WebUI**: อินเทอร์เฟซผู้ใช้บนเว็บสำหรับการเข้าถึงแบบโต้ตอบ
- **Storage Proxy**: จัดการการจัดเก็บข้อมูลถาวรสำหรับข้อมูลผู้ใช้

## รายการตรวจสอบการเริ่มต้นอย่างรวดเร็ว

เพื่อเริ่มต้นใช้ Backend.AI:

1. **เลือกวิธีการปรับใช้ของคุณ**
   - Docker Compose (แนะนำสำหรับการพัฒนา/ทดสอบ)
   - Kubernetes (แนะนำสำหรับการผลิต)
   - การติดตั้งจากซอร์ส (สำหรับผู้ใช้ขั้นสูง)

2. **ข้อกำหนดของระบบ**
   - ระบบที่ใช้ Linux (แนะนำ Ubuntu 20.04+)
   - Docker และ Docker Compose
   - RAM อย่างน้อย 4GB และ CPU 2 คอร์
   - พื้นที่จัดเก็บสำหรับอิมเมจคอนเทนเนอร์และข้อมูลผู้ใช้

3. **การตั้งค่าเริ่มต้น**
   - ติดตั้งส่วนประกอบ Backend.AI
   - กำหนดค่าการจัดเก็บและเครือข่าย
   - ตั้งค่าบัญชีผู้ใช้เริ่มต้น
   - ตรวจสอบการติดตั้ง

4. **เซสชันแรก**
   - เข้าถึง WebUI
   - สร้างเซสชันการคำนวณ
   - รันสคริปต์ Python อย่างง่าย
   - สำรวจสภาพแวดล้อมที่ใช้ได้

## ขั้นตอนถัดไป

เมื่อคุณมี Backend.AI ทำงานแล้ว:

- สำรวจ WebUI เพื่อทำความคุ้นเคยกับอินเทอร์เฟซ
- ลองสร้างเซสชันด้วยภาษาโปรแกรมที่แตกต่างกัน
- เรียนรู้เกี่ยวกับโฟลเดอร์จัดเก็บสำหรับข้อมูลถาวร
- ตั้งค่าผู้ใช้และโปรเจกต์เพิ่มเติมตามต้องการ

สำหรับคำแนะนำการติดตั้งโดยละเอียด กรุณาไปยังส่วนการติดตั้งของคู่มือนี้
