# บทนำ

ยินดีต้อนรับสู่คู่มือ Backend.AI เอกสารที่ครอบคลุมนี้ให้ข้อมูลรายละเอียดเกี่ยวกับแพลตฟอร์ม Backend.AI ครอบคลุมการติดตั้ง การกำหนดค่า การจัดการ และสถานการณ์การใช้งานขั้นสูง

Backend.AI เป็นแพลตฟอร์มโอเพ่นซอร์สที่ทำให้ทรัพยากรคอมพิวติ้ง AI เป็นประชาธิปไตยโดยการให้สภาพแวดล้อมที่ปรับขนาดได้และใช้คอนเทนเนอร์สำหรับงาน machine learning ไม่ว่าคุณจะเป็นนักวิจัย นักพัฒนา หรือผู้ดูแลระบบ คู่มือนี้จะช่วยให้คุณใช้ประโยชน์จากคุณสมบัติอันทรงพลังของ Backend.AI ได้อย่างเต็มที่

## สิ่งที่คุณจะพบในคู่มือนี้

คู่มือนี้ครอบคลุมแง่มุมต่างๆ ของ Backend.AI:

- **เริ่มต้นใช้งาน**: การตั้งค่าพื้นฐานและการกำหนดค่าเบื้องต้น
- **การติดตั้ง**: ขั้นตอนการติดตั้งโดยละเอียดสำหรับสถานการณ์การปรับใช้ที่แตกต่างกัน
- **การจัดการ**: การจัดการผู้ใช้ การจัดสรรทรัพยากร และการตรวจสอบระบบ
- **หัวข้อขั้นสูง**: สภาพแวดล้อมที่กำหนดเอง การใช้ API และรูปแบบการบูรณาการ

## ผู้อ่านเป้าหมาย

- **ผู้ดูแลระบบ**: เรียนรู้วิธีการปรับใช้และจัดการคลัสเตอร์ Backend.AI
- **นักพัฒนา**: เข้าใจการบูรณาการ API และการสร้างสภาพแวดล้อมที่กำหนดเอง
- **นักวิจัย**: ค้นพบวิธีการใช้ประโยชน์จาก Backend.AI สำหรับเวิร์กโฟลว์การวิจัย ML
- **ผู้ใช้ปลายทาง**: ทำความคุ้นเคยกับความสามารถของแพลตฟอร์มและแนวทางปฏิบัติที่ดีที่สุด

## ข้อกำหนดเบื้องต้น

ก่อนที่จะเจาะลึกคู่มือนี้ คุณควรมี:

- ความเข้าใจพื้นฐานเกี่ยวกับแนวคิดคอนเทนเนอร์ (Docker, Kubernetes)
- ความคุ้นเคยกับการจัดการระบบ Linux/Unix
- ความรู้เกี่ยวกับเวิร์กโฟลว์ machine learning (สำหรับกรณีการใช้งานวิจัย)
