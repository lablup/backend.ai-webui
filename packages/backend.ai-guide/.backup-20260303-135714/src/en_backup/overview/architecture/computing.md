# Computing Resources

Backend.AI provides flexible and scalable computing resource management to support various AI and HPC workloads.

## Resource Types

### CPU Resources
- Multi-core CPU allocation
- NUMA-aware scheduling
- CPU affinity control
- Performance monitoring

### GPU Resources
- NVIDIA GPU support
- Fractional GPU allocation (fGPU)
- Multi-GPU sessions
- GPU memory management

### Memory Management
- Dynamic memory allocation
- Memory limits and guarantees
- Shared memory support
- Memory usage monitoring

## Resource Allocation

### Static Allocation
- Fixed resource assignments
- Guaranteed resource availability
- Predictable performance
- Suitable for production workloads

### Dynamic Allocation
- On-demand resource scaling
- Efficient resource utilization
- Cost optimization
- Ideal for development and testing

## Session Types

### Interactive Sessions
- Real-time user interaction
- Jupyter notebooks
- Development environments
- Debugging capabilities

### Batch Sessions
- Automated job execution
- Scheduled processing
- Long-running computations
- Resource optimization

### Cluster Sessions
- Multi-node computing
- Distributed training
- Parallel processing
- High-performance computing

## Resource Monitoring

Backend.AI provides comprehensive monitoring:
- Real-time resource utilization
- Performance metrics
- Cost tracking
- Usage analytics
