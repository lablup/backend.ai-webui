# はじめに

Backend.AIガイドへようこそ。この包括的なドキュメントは、Backend.AIプラットフォームに関する詳細な情報を提供し、インストール、設定、管理、および高度な使用シナリオをカバーしています。

Backend.AIは、機械学習ワークロード用のスケーラブルでコンテナ化された環境を提供することで、AIコンピューティングリソースを民主化するオープンソースプラットフォームです。研究者、開発者、システム管理者のいずれであっても、このガイドはBackend.AIの強力な機能を最大限に活用するのに役立ちます。

## このガイドで扱う内容

このガイドは、Backend.AIのさまざまな側面をカバーしています：

- **はじめに**: 基本的なセットアップと初期設定
- **インストール**: さまざまなデプロイメントシナリオの詳細なインストール手順
- **管理**: ユーザー管理、リソース割り当て、システム監視
- **高度なトピック**: カスタム環境、API使用、統合パターン

## 対象読者

- **システム管理者**: Backend.AIクラスターのデプロイと管理方法を学習
- **開発者**: API統合とカスタム環境作成の理解
- **研究者**: ML研究ワークフロー用のBackend.AI活用法を発見
- **エンドユーザー**: プラットフォームの機能とベストプラクティスに慣れ親しむ

## 前提条件

このガイドを活用する前に、以下について理解している必要があります：

- コンテナ化概念の基本的な理解（Docker、Kubernetes）
- Linux/Unixシステム管理への精通
- 機械学習ワークフローの知識（研究用途の場合）
