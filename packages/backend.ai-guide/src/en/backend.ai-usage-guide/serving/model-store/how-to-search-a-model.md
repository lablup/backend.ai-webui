# How to Search a Model

The Model Store page provides several ways to find models that match your needs. You can use free-text search and filter by category, task, or label.

## Using the Search Bar

1. Navigate to the **Model Store** page from the left sidebar.
2. Type your search query in the search bar at the top of the page.
3. The model list filters in real time as you type.

The search checks the following fields for matches:

- Model title
- Description
- Task
- Category
- Labels

:::note
The search is case-insensitive. For example, searching for "llama" will match models with "Llama" or "LLAMA" in their title or description.
:::

## Using Filters

Below the search bar, three dropdown filters let you narrow down the results:

- **Category**: Filter by the broad category of models (e.g., LLM, NLP, Vision). You can select multiple categories.
- **Task**: Filter by the specific task the model performs (e.g., text-generation, question-answering). You can select multiple tasks.
- **Label**: Filter by tags or labels assigned to models. You can select multiple labels.

All filters support multiple selection. When multiple values are selected within a filter, models matching any of the selected values are shown. When filters across different fields are combined, only models matching all filter criteria are displayed.

## Refreshing the Model List

Click the reload button next to the search bar to refresh the model list from the server. This is useful when new models have been recently added to the store.

## Combining Search and Filters

You can use the text search and dropdown filters simultaneously. For example, you might type "llama" in the search bar while selecting "LLM" from the Category filter to find all LLM models related to Llama.
