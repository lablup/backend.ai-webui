# 시작하기

이 섹션은 Backend.AI 사용을 시작하는 방법에 대한 빠른 개요를 제공합니다. 새로운 설치를 설정하거나 기존 배포를 탐색하든, 이 단계들은 시작하고 실행하는 데 도움이 될 것입니다.

## 개요

Backend.AI는 여러 핵심 구성 요소로 이루어져 있습니다:

- **Manager**: 리소스와 세션을 관리하는 중앙 제어 플레인
- **Agent**: 컨테이너(세션)를 실행하는 컴퓨팅 노드
- **WebUI**: 대화형 액세스를 위한 웹 기반 사용자 인터페이스
- **Storage Proxy**: 사용자 데이터의 지속적 저장을 처리

## 빠른 시작 체크리스트

Backend.AI를 시작하려면:

1. **배포 방법 선택**
   - Docker Compose (개발/테스트 환경에 권장)
   - Kubernetes (프로덕션 환경에 권장)
   - 소스 설치 (고급 사용자용)

2. **시스템 요구사항**
   - Linux 기반 시스템 (Ubuntu 20.04+ 권장)
   - Docker 및 Docker Compose
   - 최소 4GB RAM 및 2개 CPU 코어
   - 컨테이너 이미지 및 사용자 데이터용 저장공간

3. **초기 설정**
   - Backend.AI 구성 요소 설치
   - 저장소 및 네트워킹 구성
   - 초기 사용자 계정 설정
   - 설치 검증

4. **첫 번째 세션**
   - WebUI 액세스
   - 연산 세션 생성
   - 간단한 Python 스크립트 실행
   - 사용 가능한 환경 탐색

## 다음 단계

Backend.AI가 실행되면:

- WebUI를 탐색하여 인터페이스에 익숙해지세요
- 다양한 프로그래밍 언어로 세션 생성을 시도해보세요
- 지속적인 데이터를 위한 스토리지 폴더에 대해 학습하세요
- 필요에 따라 추가 사용자 및 프로젝트를 설정하세요
